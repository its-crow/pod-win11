podcast CLI - install instructions

cli tool for downloading podcasts + youtube stuff. setup is simple.

---

pre-install (requirements)

you need these installed BEFORE using the tool:

1. python (3.10+)
   https://www.python.org/downloads/

2. pipx

    python -m pip install --user pipx
    pipx ensurepath

restart your terminal after this

3. ffmpeg (required for audio/video processing)

windows (winget):
    winget install "FFmpeg (Essentials Build)"

verify:
    ffmpeg -version

4. node.js (recommended for youtube reliability)

windows (winget):
    winget install OpenJS.NodeJS

verify:
    node -v

notes:
- yt-dlp is installed automatically by this tool
- ffmpeg is REQUIRED for video/audio merging
- node helps avoid youtube blocking issues (not strictly required, but recommended)

---

install

run this:

    pipx install git+https://github.com/its-crow/podcast.git

that installs the tool globally (but safely isolated)

---

verify it works

    podcast --help
    podcast -h

you should see the command list

---

first time setup (important)

    podcast init

this creates config + database.

folders created:
- yt downloads → ~/Videos/yt
- podcast downloads → ~/Music/pods
- app data → ~/apps

that’s it — it won’t mess with anything else on your system

---

example usage

add a podcast:

    podcast add-feed tdz https://example.com/rss.xml

refresh:

    podcast refresh --feed tdz

download latest:

    podcast download --feed tdz

---

youtube downloads

audio:

    podcast download-yt --link <url>

video:

    podcast download-yt --link <url> --video

notes:
- uses browser cookies automatically (firefox by default)
- uses node runtime if available
- helps avoid youtube blocking issues

---

updating

when i push updates:

    pipx upgrade podcast

---

uninstall

    pipx uninstall podcast

---

notes

- works from anywhere in terminal (no cd needed)
- installed in its own isolated environment (safe)
- all downloads/config stay in your home directory

---

if something breaks, just reinstall:

    pipx reinstall podcast
